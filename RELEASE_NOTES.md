# Release Notes — feature/auth-ui-improvements

> Изменения относительно ветки `develop`
> Масштаб: 360 файлов, +17 500 / −5 800 строк

---

## Backend

### Аутентификация

- Локальная авторизация — регистрация и вход по email/паролю (хеширование Argon2)
- OAuth2 — интеграция с Google и GitHub
- MFA / 2FA — TOTP-коды через `pyotp` + резервные коды
- Refresh-токены с family-based reuse detection (защита от кражи токенов)
- Токены для верификации email и сброса пароля с отслеживанием IP

### RBAC (ролевая модель)

- Новые модели: `Role`, `Group`, `UserRole`, `UserGroup`
- Admin API: управление ролями (`/admin/roles`), группами (`/admin/groups`) и пользователями (`/admin/users`)
- Декораторы-guards для проверки ролей на уровне роутов

### Хранилище файлов

- Универсальная модель `StoredFile` с полиморфными связями (`entity_type` / `entity_id`)
- Сервис хранилища: квоты, soft delete, поддержка нескольких бэкендов (local / cloud)
- Admin API: список файлов, скачивание, удаление

### База данных

- Поддержка нескольких БД: `ConnectionManager`, `EngineFactory`, `SyncEngine` (для APScheduler и OAuth)
- Мониторинг пула соединений + Grafana-дашборд + правила алертинга в Loki
- 6 новых миграций:
  - Таблицы user accounts и OAuth-аккаунтов
  - Auth-токены (refresh, email verification, password reset)
  - RBAC-таблицы (roles, groups, ассоциации)
  - MFA-поля в таблице пользователей
  - Таблица `stored_files`

## Frontend

### Страницы авторизации

- **Login** — форма email/пароль, кнопки OAuth-провайдеров, MFA-challenge (ввод 6-значного кода)
- **Signup** — регистрация нового пользователя
- **Forgot / Reset password** — flow сброса пароля

### Профиль и безопасность

- Страница профиля с редактированием данных пользователя
- Страница безопасности:
  - Управление 2FA (включить / выключить, QR-код для TOTP)
  - Список активных сессий с информацией об устройстве и датой создания
  - Отзыв отдельных сессий и всех остальных сессий сразу

### Управление доступом

- Новая вкладка **Access** в настройках — управление правами и ролями пользователей

### UI / UX

- Рефакторинг левого сайдбара: сворачиваемые секции, collapsed-режим с иконками и popup-меню, иконка chevron рядом с логотипом
- `KmDrawerLayout` — унифицированный layout-компонент для drawer-панелей (мигрировано ~23 drawer'а)
- `useDrawerResize` — composable для resizable drawer'ов
- `DetailsLayout` и `KmTabs` — унификация внутренних detail-страниц
- Индикаторы загрузки (`km-inner-loading`) добавлены на все страницы
- Рефакторинг стилей: дизайн-токены (цвета, типографика, отступы, elevation, component tokens)
