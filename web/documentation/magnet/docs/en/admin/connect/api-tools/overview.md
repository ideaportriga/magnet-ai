# API Tools

API Tools enable Magnet AI admins to establish communication with external systems like CRMs or other systems. The reason they are placed next to Knowledge Sources and Models in Magnet AI is that they are not typically used on their own, apart from Agents. However, API Tools is what really empowers Agents to solve end-user problems by communicating with external systems.

To add a new API Tool, you will need a API specification in JSON or YAML format. Let's have a closer look at API Tool configuration in the next chapter.
