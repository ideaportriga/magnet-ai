# Prompt Templates

Prompt Templates are reusable sets of instructions and LLM parameters that form the foundation of the Gen AI features in Magnet AI. 

Some use cases of Prompt Templates: 

- Control and shape output generated by the RAG Tool;
- Ensure post-processing for RAG Tools and Agents;
- Ensure the tool-calling ability for Agents;
- Execute tasks like summarization, categorization, draft message generation, etc.
