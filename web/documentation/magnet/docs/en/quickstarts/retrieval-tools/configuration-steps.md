# Steps to Configure a Retrieval Tool

Configuring a Retrieval Tool includes 5 steps:

1. Select one or multiple Knowledge Sources where content will be searched
2. Adjust advanced retrieval settings to optimize tool performance _(optional)_
3. Enable multi-lingual Retrieval _(optional)_
4. Adjust UI Settings for a polished look and feel _(optional)_
5. Test and Save.
