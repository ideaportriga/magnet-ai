# Steps to Configure a RAG Tool

Configuring a RAG Tool consists of 7 steps:

1. Select one or multiple Knowledge Sources to ground the LLM answers
2. Adjust advanced retrieval settings for most relevant content to be fetched *(optional)*
3. Review or adjust the Prompt Template for answer generation *(optional)*
4. Enable multi-lingual RAG *(optional)*
5. Enable post-processing of RAG Tool output for Usage reports *(optional)*
6. Adjust UI Settings for a polished look and feel *(optional)*
7. Test and Save.

[Read more](../../../en/admin/configure/rag-tools/configuration.md) about configuring RAG Tools.
