# Steps to Configure a RAG Tool

Configuring a RAG Tool consists of 7 steps:

1. Select one or multiple Knowledge Sources to ground the LLM answers
2. Adjust advanced retrieval settings for most relevant content to be fetched _(optional)_
3. Review or adjust the Prompt Template for answer generation _(optional)_
4. Enable multi-lingual RAG _(optional)_
5. Enable post-processing of RAG Tool output for Usage reports _(optional)_
6. Adjust UI Settings for a polished look and feel _(optional)_
7. Test and Save.

[Read more](../../../en/admin/configure/rag-tools/configuration.md) about configuring RAG Tools.
