# Meklēšanas rīki

Meklēšanas rīki nodrošina uzlabotu semantiskās meklēšanas pieredzi. Tiem ir daudz kopīga ar [RAG rīkiem](../rag-tools/overview.md): varat konfigurēt līdzības slieksni, optimizēt daudzvalodu gadījumiem, ieslēgt lietotāju atsauksmes vai pievienot pārkārtošanu tāpat kā RAG rīkiem. Galvenā atšķirība ir tā, ka meklēšanas rīki negenerē teksta atbildi uz lietotāja jautājumu. Tie vienkārši atgriež semantiski atbilstošākos satura fragmentus.

Meklēšanas rīki ir piemēroti gadījumiem, kad lietotājiem nepieciešams saņemt saites uz visatbilstošākajiem satura gabaliem, piemēram, instrukcijām vai rokasgrāmatām, un oriģinālā teksta pārfrāzēšana nesniedz papildu vērtību.
