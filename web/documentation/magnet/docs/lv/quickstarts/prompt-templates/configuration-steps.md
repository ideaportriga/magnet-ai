# Promptu veidnes konfigurēšanas soļi

Promptu veidnes konfigurēšana sastāv no 4 soļiem:

1. Uzrakstiet instrukcijas LLM
2. Pielāgojiet LLM iestatījumus *(pēc izvēles)*
3. Pielāgojiet atbildes formātu *(pēc izvēles)*
4. Testējiet un saglabājiet.

[Lasīt vairāk](../../../lv/admin/configure/prompt-templates/configuration.md) par promptu veidņu konfigurēšanu.
